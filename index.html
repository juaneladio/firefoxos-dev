Resumen: Firefox OS es un nuevo sistema operativo móvil enfocado en mercados emergentes y que desembarcó en Perú y algunos países de Latinoamérica en 2013. Lo que probablemente no sabías es que toda aplicación web puede ser llevada rápidamente a esta plataforma.
En esta presentación explicaremos porqué Firefox OS resulta atractivo a los desarrolladores, qué pasos deben seguir para portar sus aplicaciones y qué le espera en este 2014 en Perú y el resto del mundo.

Desarrollo de aplicaciones en Firefox OS

firefox.com/os

Esta es la diapositiva de introducción a Firefox OS. Explica que es un sistema operativo nuevo, basado completamente en tecnologías Web y pensado para mercados que otros no están cubriendo. Tanto cumpliendo las promesas que otros habían hecho como cumpliendo los principios definidos en el Manifiesto de Mozilla.

Un nuevo mercado

Firefox OS, y esto es importante decirlo, no se enfoca en los mismos mercados que iOS y Android. Su objetivo principal es llevar la conectividad a la web a mercados que no pueden permitirse otras plataformas o ni siquiera tienen acceso a ellas. Es es la razón por la que los usuarios de Firefox OS no estan comparando lo que realizan como desarrolladores de Aplicaciones con las últimas desarrolladas para Andoird e iOS. Tu trabajo como desarrollador de aplicaciones para Firefox OS es brindar la primera impresión de la Web a muchos nuevos usuarios. No es una tarea sencilla, pero si muy fructífera y reconfortante. En lugar de copiar Angry Birds y no tener el rendimiento de hardware que necesitas, puedes ser el nuevo Angry Birds para una audiencia totalmente nueva.

HTML5 para el mundo móvil

En el principio la promesa era que HTML5 permitiría a los desarrolladores crear aplicaciones web que le saquen provecho a los recursos de un celular. Finalmente las aplicaciones web tendrían una plataforma que no sólo estaría a la vanguardia sino también les permitiría a todos construir para estos dispositivos.

Firefox para Android

La primera respuesta de Mozilla en el frente móvil fue Firefox para Android, un navegador que llevara parte de las ventajas de Firefox de escritorio a un móvil, y que permitiera sincronizar datos de navegación entre móviles y PCs. Está disponible en Google Play y tiene más opciones que el Android Browser que viene por defecto. No hay Firefox para otras plataformas móviles por impedimentos legales.

¿Qué hace que una aplicación sea genial?

Las aplicaciones son "lo nuevo", salvo que no lo son. Esencialmente una aplicación es una cosa que realiza una sola tarea bien y es autosuficiente. En lugar de abrir un sitio web y tenerlo abierto junto a otras pestañas, abres una aplicación y es el foco principal de tu atención. Las personas aman esto. Le da al usuario una experiencia de una sola tarea sin que se distraigan. La buena noticia es que las tecnologías que rodean al HTML5 son lo suficientemente flexibles como para brindar ambos casos: puedes crear un sitio web pero también puedes convertirlo fácilmente en una aplicación.

¿Cómo empezar?

developer.mozilla.org

Hay documentación en español sobre Firefox OS en la Mozilla Developer Network.

Paso 1: Crea tu aplicación web con HTML5, CSS y JavaScript

developer.mozilla.org

Crea tu aplicación web con HTML5, CSS y JavaScript que se adecúe a los diversos tipos de pantalla.

Paso 2: Evalúa qué componentes deben ser offline.

developer.mozilla.org

Evalúa qué componentes de tu aplicación deben estar disponibles fuera de línea para ofrecer la mejor experiencia al usuario.

Paso 3: Empaqueta

developer.mozilla.org

Empaqueta. Crea un manifiesto con información de la app y otro para indicar los archivos que requiere para funcionar fuera de línea.

Paso 4: Prueba

addons.mozilla.org

Prueba tu aplicación en Firefox OS Simulator para saber cómo ejecutará en un dispositivo Firefox OS.

Paso 5: Publica

marketplace.firefox.com

Publica tu aplicación en tu sitio web, o en Mozilla Marketplace para su distribución gratuita o de pago.
